# 好物甄选

> 本仓库仅用于学习，部分图片素材 和 UI 设计取自京东，请勿直接用于商业活动。

基于 [vitejs][vitejs] + [vue3][vue-next] + [vant][vant] 搭建的在线商城，目前正在积极开发中~。
